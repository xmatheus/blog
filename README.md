### Getting Started

run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

or

run the production server

```bash
npm run build; npm run start
# or
yarn build; yarn start
```

Open [http://localhost:3000](http://localhost:3000).
